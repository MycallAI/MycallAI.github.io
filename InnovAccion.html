➤ Develop a Communication Solutions Prototype for Rural Areas

This project directly aligns with MYCALLA INC.'s purpose to "empower and connect the people of Puerto Rico through innovative and reliable communication solutions," specifically focusing on bridging gaps. A junior engineer could contribute by researching existing low-cost communication technologies (e.g., LoRaWAN, mesh networking, or satellite internet solutions for underserved areas) and developing a basic proof-of-concept or a simulated model of how these technologies could be implemented in a rural Puerto Rican community to extend network access or provide emergency communication.

➤ Leverage Juniors for Internal ROI: Build a "Network Health Dashboard" Internal Tool

Instead of a broad external project, a junior engineer can focus on an internal tool. This project involves developing a dashboard that monitors key performance indicators (KPIs) of MYCALLA INC.'s communication network. The junior engineer can utilize their programming skills (e.g., Python for data analysis, as per your background) to collect, process, and visualize data related to network uptime, latency, bandwidth usage, and user experience. This provides real-world experience in data management and system monitoring, while giving the company direct value by offering insights into network performance and identifying areas for improvement.

➤ Build a Real Talent Pipeline: Document Best Practices and Create Training Modules

While working on the network health dashboard or communication prototype, the junior engineer can be tasked with documenting the development process, coding standards, and best practices. This output can then be used to create internal training modules for future junior hires or interns, thereby contributing to MYCALLA INC.'s talent pipeline. This allows the company to "stop waiting for talent to fall from the sky" and instead build an effective onboarding process, attracting and developing the "best talent on the island."
